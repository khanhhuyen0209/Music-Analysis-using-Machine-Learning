# Music-Analysis-using-Machine-Learning
Chord: Phân loại các hợp âm sử dụng MLP 2-hidden-layer
Pitch: Phân loại các quãng âm sử dụng MLP 2-hidden-layer
Tempo: Phân loại các tempo sử dụng Logistic Regression
Genre: Phân loại các thể loại âm nhạc sử dụng MLP 2-hidden-layer
Emotion: Phân loại cảm xúc của đoạn nhạc dựa trên các feature: genre, chord, pitch, tempo, .. sử dụng kết quả từ model đã train trước đó
ESP32: Chạy test các model trên esp32 sử dụng mic thu âm
